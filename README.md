# Frontend PolyQuiz

## Description

Frontend de l'application PolyQuiz développé avec React.

L'application permet aux utilisateurs de :

* Consulter les quiz disponibles
* Répondre aux questions
* Visualiser leurs résultats
* Interagir avec le backend via une API REST

## Technologies utilisées

* React
* Vite
* JavaScript
* CSS
* Axios

## Installation


### Installer les dépendances

```bash
npm install
```

### Lancer l'application

```bash
npm run dev
```

## Accès à l'application

Une fois démarrée :

```text
http://localhost:5173
```

Le frontend communique avec l'API PolyQuiz via :

```text
http://localhost:5000/api
```
